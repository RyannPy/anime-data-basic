# Anime Data Analysis

## Deskripsi

Project ini bertujuan untuk menganalisis data anime menggunakan Python dan pandas.

## Tujuan

- Menampilkan data anime dari file CSV
- Melakukan filtering berdasarkan rating dan tahun rilis
- Menghitung statistik dasar (mean, median, min, max)
- Membandingkan rating sebelum dan setelah tahun 2000

## Dataset

Dataset `anime.csv` berisi informasi:

- Judul anime
- Rating
- Tahun rilis

## Tools yang digunakan

- Python
- pandas
- Jupyter Notebook

## Analisis yang dilakukan

- Filter rating (rendah, sedang, tinggi)
- Filter berdasarkan tahun rilis
- Perhitungan statistik
- Perbandingan rata-rata rating

## Insight

Rata-rata rating anime setelah tahun 2000 lebih tinggi dibanding sebelum tahun 2000. Hal ini kemungkinan dipengaruhi oleh perkembangan kualitas animasi dan industri anime dan juga perkembangan selera penggemar, dimana selain jalan cerita, kualitas visual dan audio menjadi salah satu faktor utama penilaian kepuasan.

## Cara Menjalankan

1. Install Python
2. Install library: pip install pandas jupyter
3. Jalankan file `analysis.ipynb`
